# Snake-Game
🐍 A feature-rich Snake Game built with HTML, CSS, and JavaScript featuring a 3-life system ❤️, smooth animations, sound effects, background music, pause/resume, high score tracking, responsive UI, and polished gameplay experience.

---

## 🎮 Features

- 🐍 Classic Snake Gameplay
- ❤️ 3 Lives System
- 🍎 Random Food Generation
- 📈 High Score saved using Local Storage
- ⏱️ Live Timer
- 🔊 Sound Effects & Background Music
- 🎚️ Volume Control
- ⏸️ Pause / Resume Game
- 💀 Game Over Screen
- ✨ Modern Neon UI
- 📱 Responsive Design
